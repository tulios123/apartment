# שפת-עיצוב — "ניהול דירה"
(כל הערכים מתוך `src/index.css`. אין UI framework — CSS ידני עם design tokens.)

## פלטת-צבעים (ברירת-מחדל, מצב בהיר)
- **אקסנט (פעולה ראשית):** כחול `#2a7de1` (hover `#1f66c0`, רקע-בהיר `#e8f1fc`). פלטה בהשראת "ANZ Plus".
- **הצלחה / חיובי:** ירוק `#1fa871` (רקע `#dcf3ea`, טקסט `#127a52`).
- **סכנה / שלילי:** אדום-אלמוגי `#e0533d` (רקע `#fde6e1`).
- **אזהרה:** חום-כתום `#b06a00` (רקע `#fdf2dc`).
- **navy של כותרות וכרטיסי-Hero:** `#0a1f44`.
- **דקורטיבי / גרפים:** סגול `#6d4ad6`, אלמוג `#f0654e`, טורקיז `#2bb39b`, sage.
- **משטחים:** רקע `#f5f6f8`, כרטיס/משטח `#ffffff`, גבול `#e3e7ec`.
- **טקסט:** ראשי `#1a1a1a`, עמום `#6e7480`.

## טיפוגרפיה
- גופן יחיד: **Heebo** (Google Fonts), משקלים 400–800 — גם לגוף וגם לתצוגה. נפילה: system-ui.
- בסיס: 15px / גובה-שורה 1.6. כותרת-עמוד 22px/700.
- **ספרות ברוחב-אחיד (`tabular-nums`)** באופן גלובלי — מיישר סכומים בטורים (חשוב לאפליקציה פיננסית), משפיע על ספרות בלבד.

## מרווחים, רדיוסים, צללים, מעברים
- **סולם-מרווחים (קצב 8px):** 4, 8, 12, 16, 20, 24, 32, 40 (`--s-1`…`--s-10`).
- **רדיוסים:** 6 / 8 / 16 / 20 / 999 (pill). ברירת-מחדל 16.
- **צללים:** sm / md / lg (גוון navy עדין).
- **מעברים:** 150 / 200 / 300ms, easing `cubic-bezier(0.4,0,0.2,1)`.

## דפוסים חוזרים
- **כפתור ראשי:** רקע אקסנט-כחול, טקסט לבן, פינות-pill, ריפוד ~9×20px.
- **כרטיס:** רקע לבן, רדיוס 16, צל-md, גבול עדין.
- **Hero (כרטיס-נתון מרכזי):** רקע navy `#0a1f44`, טקסט לבן.
- **ניווט:** sidebar ימני ברוחב 210px בדסקטופ; bottom-nav במובייל (≤640px); drawer נשלף לפריסות ביניים.

## מצב כהה
מיושם דרך `:root[data-theme="dark"]` — **החלפת tokens מלאה** (משטחים כהים בגוון slate-navy, אקסנטים מוארים מעט, navy-הטקסט מתבהר בעוד כרטיסי-Hero נשארים navy). `src/lib/theme.ts` קובע את `data-theme` לפי בחירת המשתמש (בהיר / כהה / לפי-המכשיר), ו-`color-scheme` מסונכרן כדי שפקדים native (תאריך/בחירה) יתאימו לערכה.

## מימין-לשמאל (RTL)
- האפליקציה עברית, מימין-לשמאל לכל אורכה.
- ה-sidebar נעוץ לקצה **ימין**; ה-drawer במובייל נכנס **מימין**; שברון-החזרה באשף יושב **ימני-עליון** (RTL: "אחורה" = לכיוון ההתחלה = ימין).
- בוטל overscroll-bounce (חשוב במצב PWA עצמאי).
