# סקירת-מוצר — "ניהול דירה"

## מה זה ולמי
אפליקציית ווב (PWA) בעברית, מימין-לשמאל, מותאמת-מובייל, לניהול השקעה בדירה אחת מושכרת. מיועדת למשקיע פרטי (ובהמשך לבני משפחה — כל אחד מנהל דירה משלו), שרוצה לראות במקום אחד: תזרים חודשי, הון עצמי וצבירת-בעלות, מבנה המימון (משכנתא והלוואות), חוזי-שכירות, ביטוח, משימות ומסמכים.

## סטאק טכני (בקצרה)
React 19 + TypeScript + Vite; Supabase (בסיס-נתונים Postgres + הזדהות + אחסון קבצים); React Router; אייקוני Phosphor; גרפים ב-SVG ידני (בלי ספריית-גרפים); בלי UI framework — CSS ידני עם design tokens. PWA עם Service Worker (התראות push, מניפסט).

## מצב נוכחי
האפליקציה פעילה ובשימוש אמיתי של הבעלים. הליבה הפיננסית מכוסה בבדיקות אוטומטיות (Vitest). היא בנויה סביב **4 עמודי-תווך** (ראשי / תזרים / הון / ניהול-הנכס) + הגדרות.
*בתהליך (לפי מסמכי-הסקירה בריפו, לא נגזר מהקוד לבדו):* הכנה לשיתוף עם בני המשפחה — פרסום ההזדהות דרך Google, מעבר משתמש-חדש מאפס, ליטושי חוויית-משתמש, ואמינות/גיבוי.

## פיצ'רים מרכזיים שכבר קיימים (מאומת בקוד)
- **Onboarding** — אשף 9 שלבים להקמת הנכס, המימון, השכירות והביטוח; כולל העלאת מסמכים וחילוץ-נתונים אוטומטי (AI) ממסמכי-בנק/חוזה (edge functions).
- **ראשי (Home)** — ברכה לפי שעה, "מרכז פעולות" (אישור גביית שכר-דירה, משימות פתוחות, התראות חידוש-חוזה), קליטה-מהירה בטקסט חופשי + כפתורי הוצאה/משימה, וכרטיס תזרים חודשי (צפי לסוף החודש).
- **תזרים (Finances)** — רשימת תנועות (הכנסה/הוצאה) עם הוספה/עריכה/מחיקה, שורות-תחזית אוטומטיות (שכר-דירה, משכנתא, ביטוח, הלוואות) המסומנות "קבוע"/"תחזית", תצוגת חודש/שנה/טווח, פירוט לפי קטגוריות, צירוף-קבלה.
- **הון (Wealth)** — מצב-בעלות (הון עצמי מול חוב-בנק מול בלון, אחוז-בעלות), צבירת-קרן חודשית, תשואה, ועורך-מימון מוטמע (מסלולי-משכנתא + הלוואות + עלויות-השקעה).
- **ניהול הנכס** — כריכת-נכס (כתובת/שווי/תג חוזה-פעיל) + 4 טאבים: חוזה, ביטוח, משימות, מסמכים.
- **משימות** — רשימה עם סימון-בוצע; השלמת משימה מציעה לרשום תנועה כספית מתאימה (תיקון→הוצאה, שכר→הכנסה).
- **מסמכים** — העלאה ואחסון לפי סוג (חוזה/ביטוח/משכנתא/קבלה...), קישור לישויות, גישה מאובטחת (bucket פרטי).
- **התראות push** — תזכורות יומיות (אישורים, חידושי-חוזה, משימות באיחור).
- **הגדרות** — חשבון (Google/אימייל, יציאה), מראה (בהיר/כהה/לפי-המכשיר), התראות, וכלי-מנהל מוגנים.
- **מתמטיקת-משכנתא** — חישוב שפיצר, ריבוי-מסלולים (פריים+מרווח, תקופות-גרייס), לוח-סילוקין.
- **חוסן** — Error boundaries (כללי + לכל מסך), באנר לא-מקוון, retry על שגיאת-טעינה.
