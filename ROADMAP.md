# המפה — רשימת-העדיפויות החיה

> מקור-האמת היחיד למה עושים עכשיו, מה הבא ומה בהמשך.
>
> **כללי שימוש:**
> - כל פריט מתויג: 🟢 **בטוח** (קלוד מבצע ודוחף לבד) / 🔴 **דורש-בעלים** (הכנה בלבד, עצירה להחלטה).
> - אחריות: [קלוד] / [בעלים] / [שנינו].
> - פריט עובר ל"בוצע" **רק** עם שורת "איך אומת:" לצידו — בדיקה בפועל, לא זיכרון.
> - פתיחת סשן: `/kickoff` (מושך מגיט וקורא מכאן) · סיום סשן: `/wrap` (מעדכן כאן).

## 🎯 עכשיו — שחרור למשפחה

- [ ] 🔴 [שנינו] **ולידציית זיהוי-המסמכים על מסמכים מבנקים ופורמטים אחרים** — פריט-הסיכון מספר אחת לשחרור. הבעלים מביא מסמכים אמיתיים של בני המשפחה (משכנתא, רכישה, שכירות — מבנקים שונים), קלוד מריץ את החילוץ, מודד מול האמת ומתקן. _השפעה: גבוהה מאוד · מאמץ: בינוני._
- [ ] 🔴 [בעלים] **קליטת ממצאים קיימים** — ההערות מהרישום בטלפון של האחות + ממצאי המעבר של הבעלים מהשבוע. כל הערה הופכת לפריט-וידוא נפרד במפה, ומסומנת בוצע רק אחרי בדיקה בפועל — לא לפי זיכרון. _השפעה: גבוהה · מאמץ: קטן._
- [ ] 🔴 [בעלים] **פרסום ההזדהות של גוגל לסביבת-אמת** — במסוף-הענן של גוגל: מסך-ההסכמה ממצב "בדיקות" למצב "ייצור", כדי שכל בן-משפחה יוכל להתחבר. _השפעה: גבוהה (חוסם כניסה) · מאמץ: זעיר._
- [ ] 🔴 [שנינו] **מעבר משתמש-חדש מאפס** — הרשמה בחשבון חדש אמיתי, מעבר על האשף ועל כל מסך ריק ברוחב טלפון, תיקון מה שנשבר. _השפעה: גבוהה · מאמץ: בינוני._
- [ ] 🟢 [קלוד] **מדריך "הוסף למסך-הבית + הפעל התראות"** — הסבר פשוט לבני-משפחה (בייחוד אייפון: התראות דורשות התקנה למסך-הבית). _השפעה: בינונית-גבוהה · מאמץ: בינוני._
- [ ] 🔴 [שנינו] **אמינות** — וידוא שכל המיגרציות חלות בפועל, שתזמון-התזכורות היומי רץ, בירור מדיניות-הגיבויים, ודרך להבחין בתקלה. _השפעה: גבוהה · מאמץ: קטן-בינוני._

### דגשי הסקירה מהצ׳אט
_(הסשן המקביל מטפל בחלק מהדגשים במספור א1, א2… — לפני התחלת פריט מכאן, לוודא מול הקומיטים שהוא לא כבר בעבודה.)_

- [ ] 🟢 [קלוד] **ניקוי שכבת-המפתח** — כניסת-המנהל בלוגין, כפתור-הפידבק עם המונה, ואזורי-המנהל בהגדרות (בדגש על איפוס-הנתונים ההרסני) נעלמים לחלוטין למי שאינו המנהל. איך מאמתים: כניסה כמשתמש רגיל — אפס זכר לכלי-מפתח.
- [ ] 🟢 [קלוד] **עברית מלאה בחזית** — הכותרת "Apartment", שם-האפליקציה במניפסט (מה שמופיע מתחת לאייקון אחרי התקנה), והודעות-שגיאה שמגיעות מהשרת. _השפעה: גבוהה · מאמץ: נמוך._
- [ ] 🟢 [קלוד] **מצבי-ריק אחרי "סיים עכשיו"** — משתמש שדילג באונבורדינג על משכנתא/חוזה/ביטוח: כל אחד מ-4 העמודים מציג מצב-ריק מכוון עם כפתור-השלמה, לא מסך עם אפסים שנראה שבור.
- [ ] 🔴 [שנינו] **גילוי-נאות למסלול צמוד** — ההצמדה מחושבת נומינלית בלבד; בן-משפחה עם מסלול צמוד יראה בשקט מספרים שנשחקים מהאמת. החלטה נדרשת: תווית "ללא הצמדה למדד" עכשיו, מימוש מלא בהמשך-הדרך.
- [ ] 🟢 [קלוד] **סירוב-התראות לא שובר כלום** — מי שלוחץ "לא לאשר" על הפוש ממשיך לעבוד רגיל, ויש נתיב ברור להפעלה-מחדש מההגדרות.

## 🎯 עכשיו — מנוע-הכיסוי (סקירות)
_כל סקירה מייצרת ממצאים שנכנסים כפריטים חדשים למפה. מתאימות לעבודת-לילה._

- [ ] 🟢 [קלוד] **סקירת אבטחה ומקרי-קצה** — הרשאות, בידוד-נתונים, קלטים חריגים, מצבי-כשל.
- [ ] 🟢 [קלוד] **סקירת דיוק עיצובי וחוויית-משתמש** — עקביות מרווחים/צבעים/רכיבים, זרימות, מצבים ריקים.
- [ ] 🟢 [קלוד] **סקירת פונקציונליות ובדיקות** — כל יכולת מקצה-לקצה, הרחבת כיסוי-הבדיקות.
- [ ] 🟢 [קלוד] **סקירת מוכנות-לשחרור** — מעבר על הכול בעיניים של בן-משפחה לא-טכני.

## ⏭ הבא
- דוח-מס שנתי (ייצוא מסמך).
- מוקד חידוש-חוזה.
- ניקוי המסמכים המיושנים בריפו — מיזוג מה שעוד חי לכאן, סימון השאר כארכיון.

## 💡 בהמשך
- סימולטור-תרחישים פיננסי — מכירה מול החזקה, שינויי-ריבית, מיחזור, פירעון מוקדם.
- ערכים ריאליים — הצגה מתואמת-מדד: ריבית ריאלית, תשואה ריאלית, שווי בשקלים של היום.
- משיכה אוטומטית של מדד-המחירים וריבית בנק-ישראל.
- מסך "סגירת חודש".

## ✅ בוצע (מתגלגל)
- **הקמת מערכת-העבודה** — המפה הזו, דלת-הכניסה, טקסי הפתיחה/הסיום, וחבילת-ההסבר (docs/handoff). איך אומת: הקבצים בריפו; בדיקת טקס-הפתיחה בסשן חדש — ממתינה.
